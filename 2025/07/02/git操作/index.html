Git 操作指南

一、Git 基础概念

  • 仓库：Git 是一个分布式版本控制系统，用于管理项目的版本。仓库是存储项目代码和版本信息的地方，分为本地仓库和远程仓库。

  • 仓库（Repository）

    • 本地仓库：存在本机的代码版本库（.git目录）。
    • 远程仓库：托管在 GitHub/GitLab 等平台的代码库（如 origin）。

  • 三大区域

    • 工作区（WorkSpace）：你直接编辑的文件（未提交的修改）。
    • 暂存区（Staging Area）：通过 git add 暂存的文件（准备提交），工作区的**.git隐藏文件夹下**。
    • 版本库（Repository）：通过 git commit 提交的永久快照。工作区的**.git隐藏文件夹下**的HEAD指向的最终版本。

  • 分支（Branch）

    • 每个分支代表独立的开发线，默认主分支为 main 或 master。
