1.HTML

1.1.基本内容与标签

  • <html>标签：是 HTML 文档的根标签，所有其他 HTML 元素都嵌套在其中。它有两个主要子标签：<head>和<body>。

  • <head>标签：包含文档的元数据，这些信息不会直接在页面内容中显示，但对浏览器和搜索引擎很重要。

  • <body>标签：包含了网页中所有可见的内容，如文本、图像、链接、表格、表单等。

六、布局标签div和span(分区和标记文本)

  1. div
  • <div>是 HTML 中的一个块级元素，“div” 是 “division”（分区、划分）的缩写。它本身没有特定的语义，主要用于将 HTML 文档划分为不同的逻辑部分，作为其他 HTML 元素的容器。

它有一个开始标签<div>和一个结束标签</div>，在这两个标签之间可以包含各种其他的 HTML 元素，如文本、图像、链接、列表、表格等。例如：

  • 布局方面

    • 在网页布局设计中，<div>元素是构建页面结构的重要工具。通过使用多个<div>元素，可以将页面划分为不同的区域，如头部（header）、导航栏（navigation）、主体内容（content）、侧边栏（sidebar）和页脚（footer）等。例如，在一个典型的网页布局中：

1.2.常用标签扩展

  1. article
  • 用于表示文档、页面或者网站中独立的、完整的、可被外部引用的内容块。它所包裹的内容通常自身具有一定的主题性和独立性，即便脱离了所在的网页上下文，也依然能够被清晰地理解和单独传播。例如一篇博客文章、一则新闻报道、一个论坛帖子、一个用户评论等，都可以使用 <article> 标签来进行标记。

  • 整个博客文章内容放在 <article> 标签里，文章有标题（在 <header> 中）、正文内容、不同的章节（通过 <section> 划分）以及结尾的提示语（在 <footer> 中），结构清晰且符合语义化要求。

  • <article> 标签常常会和其他语义化标签一起配合使用，来进一步完善内容的结构：

    • **<header>**：用于定义 <article> 的头部信息，通常包含文章的标题、作者、发布时间等元信息。
    • **<section>**：可以用来划分 <article> 内部的不同章节或逻辑部分，让内容的组织更有条理。
    • **<footer>**：一般放置与该 <article> 相关的补充信息，比如版权声明、相关链接、评论引导等内容。

    使用 <article> 标签有助于提升网页的语义化程度，使得网页结构更清晰，对于搜索引擎优化（SEO）、无障碍访问（如屏幕阅读器更好地解析内容）等方面都有积极作用。

2.1.Vue概述

  • Vue是一套前端框架，免除原生JavaScript中的DOM操作，简化书写。
  • 基于MVVM(Moder-View-ViewMode)思想，实现数据的双向绑定，将编程的关注点放在数据上。
  • 框架：是一个半成品软件；是一套可重用的、通用的、软件基础代码模型。基于框架进行开发，更加快捷、更加高效。
