SQL（Structured Query Language）即结构化查询语言，用于管理关系型数据库。以下是 SQL 的主要语法：

一、数据查询语言（DQL）

主要用于从数据库中检索数据。

  1. SELECT

    语句

    • 基本用法：用于选择表中的列。
  • 选择所有列：可以使用通配符 “*”。
  • 给列起别名：使用AS关键字。

  1. WHERE

子句

  • 用于筛选满足特定条件的行。
  • 条件可以是比较运算符（如=, <, >, <=, >=, <>）、逻辑运算符（如AND, OR, NOT）等的组合。
  1. ORDER BY
  • 默认是升序（ASC），可以指定降序（DESC）。
  1. LIMIT
  • 也可以指定起始位置和行数，如LIMIT offset, row_count。

二、数据操作语言（DML）

用于对数据库中的数据进行插入、更新和删除操作。

在 SQL 中，为了确保兼容性和遵循标准，应该使用单引号来表示字符串常量。而双引号通常用于特定数据库管理系统中的对象命名约定。

  1. INSERT
  • 插入多行数据：可以使用多个VALUES子句。
  1. UPDATE



  2. DELETE

三、数据定义语言（DDL）

用于定义数据库对象，如表、索引、视图等。

  1. CREATE TABLE
  • 数据类型可以是整数（如INT）、字符串（如VARCHAR）、日期（如DATE）等。
  • 约束可以是主键（PRIMARY KEY）、外键（FOREIGN KEY）、唯一约束（UNIQUE）等。

四、数据控制语言（DCL）

用于控制数据库的访问权限和事务管理。

  1. GRANT
  • 权限可以是SELECT, INSERT, UPDATE, DELETE, CREATE, DROP等。

这只是 SQL 语法的一个简要概述，实际应用中还有很多高级特性和扩展。不同的数据库管理系统可能会有一些特定的语法差异。
