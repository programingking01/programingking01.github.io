算法

1.位运算（不会改变原先的数）

1.1按位与

符号： &

对应位都为一时得一

1.2按位或

符号： |

对应位有一个为一得一

1.3按位异或

符号： ^

对应位不同时得一

例：一组数中只有一个数只有一个，其余的数都有两个，找出这个数。

1.4左移

符号： <<

左移一位相当于乘以二

1.5右移

符号： >>

右移一位相当于除以二

  1. a^b：
  • 这是一个表达式，计算 a 和 b 的按位异或结果。这个结果如果不被存储或使用，通常不会产生实际的影响。
  • 例如：
  1. a^b = a：
  • 这是一个赋值语句，将 a 和 b 的按位异或结果重新赋值给 a。这会修改 a 的值，而 b 的值保持不变。

总结：

  • a^b a>>1; 只是计算按位异或结果/右移 不修改任何变量的值。
  • a^b = a; a>>=1 修改 a 的值为 a 和 b 的按位异或结果/a右移的结果。
  • a^b = b; 和 a^b = 1; 都是错误的语法，会导致编译器报错。

一、增加元素

  1. push_back()函数
    • 功能：在vector的末尾添加一个元素。
    • 示例：
      • vector<int> v;
      • v.push_back(1);
      • v.push_back(2);
      • 此时v中包含元素1和2。
  2. insert()函数
    • 功能：在指定位置插入元素。
    • 单元素插入：
      • 语法：v.insert(it, value);，其中it是迭代器指向插入位置，value是要插入的元素。
      • 示例：
        • vector<int> v = {1, 3};
        • auto it = v.begin() + 1;
        • v.insert(it, 2);
        • 操作后v变为{1, 2, 3}。
    • 多元素插入：
      • 语法：v.insert(it, n, value);，n是要插入的相同元素value的数量。
      • 示例：
        • vector<int> v = {1, 3};
        • auto it = v.begin() + 1;
        • v.insert(it, 2, 2);
        • 操作后v变为{1, 2, 2, 3}。
    • 范围插入：
      • 语法：v.insert(it, first, last);，first和last是另一个容器的迭代器，表示要插入的元素范围。
      • 示例：
        • vector<int> v1 = {1, 2};
        • vector<int> v2 = {3, 4};
        • auto it = v1.begin() + 1;
        • v1.insert(it, v2.begin(), v2.end());
        • 操作后v1变为{1, 3, 4, 2}。

二、删除元素

  1. pop_back()函数
    • 功能：删除vector末尾的一个元素。
    • 示例：
      • vector<int> v = {1, 2, 3};
      • v.pop_back();
      • 操作后v变为{1, 2}。
  2. erase()函数
    • 功能：删除指定位置的元素。
    • 单元素删除：
      • 语法：v.erase(it);，it是指向要删除元素的迭代器。
      • 示例：
        • vector<int> v = {1, 2, 3};
        • auto it = v.begin() + 1;
        • v.erase(it);
        • 操作后v变为{1, 3}。
    • 范围删除：
      • 语法：v.erase(it1, it2);，删除迭代器it1和it2之间（不包括it2）的元素。
      • 示例：
        • vector<int> v = {1, 2, 3, 4};
        • auto it1 = v.begin() + 1;
        • auto it2 = v.begin() + 3;
        • v.erase(it1, it2);
        • 操作后v变为{1, 4}。
  3. clear()函数
    • 功能：清空vector中的所有元素。
    • 示例：
      • vector<int> v = {1, 2, 3};
      • v.clear();
      • 操作后v变为空。

三、修改元素

  1. 通过下标修改
    • 示例：
      • vector<int> v = {1, 2, 3};
      • v[1]=5;
      • 操作后v变为{1, 5, 3}。
  2. 通过迭代器修改
    • 示例：
      • vector<int> v = {1, 2, 3};
      • auto it = v.begin() + 1;
      • *it = 5;
      • 操作后v变为{1, 5, 3}。

四、查询元素

  1. 通过下标查询
    • 示例：
      • vector<int> v = {1, 2, 3};
      • int a = v[1];
      • 这里a的值为2。
  2. 通过at()函数查询
    • 示例：
      • vector<int> v = {1, 2, 3};
      • int b = v.at(1);
      • 这里b的值为2，不过at()函数会进行边界检查，如果下标越界会抛出out_of_range异常。
  3. 通过迭代器遍历查询
    • 示例：
      • vector<int> v = {1, 2, 3};
      • for (auto it = v.begin(); it!= v.end(); ++it) {
      • cout << *it << " ";
      • }
      • 这种方式会依次输出vector中的元素1、2、3。

2.寻找两个正序数组的中位数(算法的时间复杂度为log（m+n）)

方法一：暴力把数组二放到数组一中并且sort排序，然后求中位数

时间复杂度：nlogn(快排)没有利用数组有序的条件

方法二：暴力二：逆向双指针重新得到一个数组，然后得出中位数

时间复杂度：O(N)

方法三：双指针法（不创建数组）使用双指针直接找到中位数

思路：利用奇数偶数的特性，奇数数组size/2，对应的正好是中位数，而偶数数组size/2，对应右边的中位数，定义一左一右，用来求中位数，奇数时取右边的，偶数时取平均值。再用双指针每次得出最小的数，并且在下次找数之前让左=右，直到索引到达n+m/2的位置，此时由左，右值求中位数

时间复杂度：O（m+n）
