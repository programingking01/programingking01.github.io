4.1如何将配置文件作为资源文件添加到项目中

QT新建（Qt Widgets Application）

TCP Client(Tcp模式下的客户端)

  1. 在 qt中创建一个TcpClient文件。
  2. 在项目文件目录下创建一个client.config文件。（配置文件）
  3. 用txt打开文件写入本地环回ip和端口（127.0.0.1和8888）
  4. 将该文件作为资源文件添加到项目中-添加新文件-Qt-Qt Resource File
  5. 命名为config,添加前缀（区别于其它资源文件）这里用一个/。
  6. 添加文件：选client.config,并且保存一下。
